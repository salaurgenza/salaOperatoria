# Gestione Sale Operatorie — Dashboard

Apri `index.html` in un browser moderno. Include:
- Planner orario (08:00–20:00)
- Drawer con timeline chirurgica, équipe e PDF
- KPI cards dinamiche
- Assegnazioni équipe (lista)
- 5 grafici (Chart.js) dell'occupazione

Dipendenze CDN:
- Bootstrap 5.3.3
- Chart.js
- jsPDF

Per test rapido, puoi decommentare il blocco "Fake sample" nel JS per popolare un caso di esempio.
